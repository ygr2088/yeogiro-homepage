
# CLAUDE.md
> Project: 여기로양한방동물병원 디지털 플랫폼

## 목적
이 문서는 Claude Code가 프로젝트의 철학, 디자인, 개발 규칙을 항상 동일하게 이해하도록 하기 위한 기준 문서이다.

---

# 1. 프로젝트 비전

여기로양한방동물병원은 일반 동물병원이 아니라 **한방재활 특화 양한방 통합 동물병원**이다.

모든 개발과 콘텐츠는 아래 목표를 따른다.

- 보호자가 신뢰하는 브랜드 구축
- 한방재활 전문성 강조
- 실제 진료 기반 콘텐츠
- 모바일 중심 UX
- 장기적으로 AI 기반 디지털 플랫폼 구축

# 3. 병원 강점

- 20년 임상경력
- 34,000건 이상 진료
- 2,800건 이상 치과수술
- 건국대학교·경북대학교 외과 석사 의료진 2인 협진
- 치과 원장 출신 의료진
- 외과·한방재활 원장 출신 의료진
- 한방재활 인증 수의사
- EKICO 보행분석
- 양한방 통합 진료
